倒数日插件使用说明:  
/add_countdown <名称> <日期YYYY-MM-DD> [备注] - 添加倒数日

/del_countdown <名称或序号> - 删除倒数日

/countdown - 显示所有倒数日  

/countdown <名称> - 显示特定的倒数日与进度
  

/countdown_help - 显示帮助信息  


示例:  
/addcountdown 生日 2025-12-31 我的生日  

/countdown list  

/countdown delete 1
